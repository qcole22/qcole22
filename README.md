# Quentin — DevOps Projects

A focused collection of personal projects showcasing practical DevOps work: CI/CD pipelines, infrastructure as code, containerization, and orchestration. Clear, repeatable patterns and documentation for deployment and automation.

## Skills
- Cloud: AWS, Azure (learning and applied)
- Containerization: Docker
- Orchestration: Kubernetes
- Infrastructure as Code: Terraform
- CI/CD: Pipeline design and implementation (GitHub Actions, GitLab CI, etc.)
- Tools: Helm, Git, Bash, YAML

## Selected Projects
- CI/CD pipeline examples — reusable pipeline templates and best practices for automated testing, build, and deploy.
- Terraform infra modules — modular, documented modules for VPC, networking, and basic compute.
- Containerized apps + Kubernetes — sample apps with Dockerfiles, Helm charts, and deployment manifests.

## Practices
- Small, focused repos per project.
- Clear README and usage steps for each project.
- Emphasis on observability, secrets handling, and repeatable deployments.

## How to use this collection
1. Browse each project folder.
2. Read the project README for prerequisites and deployment steps.
3. Use the provided Terraform modules and pipeline templates as a starting point for your own environment.
